# 小明 (Xiaoming)

小明 (Xiaoming) 是一套专为编码代理（Coding Agents）设计的完整软件开发方法论，它建立在可组合的技能集以及一系列引导代理必须使用 these 技能的初始指令基础之上。


## 安装指南 (Cursor)

```bash
# 克隆后安装
git clone https://github.com/hj1003862396/xiaoming.git
cd xiaoming
./scripts/sync-to-cursor-local.sh
```

```text
# 启用插件
Cursor Settings → Plugins → 启用 Xiaoming
```

```bash
# 更新
./scripts/sync-to-cursor-local.sh
```

```bash
# 验证
./scripts/verify-xiaoming.sh
```

```text
# 验收
/xiaoming 你好
```
